# Unterputz Netzteil Platine
## benötigte Hardware 

* 1x Netzteil MEAN WELL IRM-05-5 oder HI-Link HLK-PM01
* 2x Federkraftklemme 2-pol, Ø 1,5 mm, RM 3,5  (AST 02 3,5) 

# Achtung: 
Ich möchte jedoch eindringlichst darauf hinweisen, 
dass beim Verbau der HI Link Netzteile ohne Luftzirkulation (Unterputz) unbedingt eine Thermosicherung mit integriert werden sollte!
Weiterhin sind immernoch Fake Netzteile im Umlauf. [Link](https://github.com/ronnythomas/Unterputz-Netzteil-Platine/issues/1)
